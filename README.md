# Huang Yutong - Personal Website Prototype

这是一个可交互的个人网页原型，展示了以下设计概念：

## 📋 包含的板块

1. **Hero Section（首屏）**
   - 名字粒子悬停特效
   - 打字机动画展示多重身份
   - 教育背景与 GPA 展示

2. **Skills Compass（技能罗盘）**
   - 双核技能卡片（数据 & 产品）
   - 交互式雷达图
   - 动态技能条动画

3. **Project Archive（项目档案）**
   - 悬浮卡片效果
   - MCM、海南旅游、台风预警项目展示

4. **Journey Timeline（经历时间轴）**
   - 三段实习经历
   - 玻璃拟态卡片

5. **Creative B-Side（创意B面）**
   - 媒体与视觉工作
   - 可展开的荣誉徽章

## 🎨 设计特点

- **玻璃拟态 (Glassmorphism)** - 磨砂玻璃材质
- **科技极简风** - 冷色调配色
- **微交互动画** - 悬停、滚动触发动画
- **粒子背景** - 数据点网络效果
- **响应式设计** - 支持移动端

## 🚀 使用方法

直接用浏览器打开 `index.html` 即可预览

如需转为正式项目，可使用以下技术栈：

- **Next.js** - React 框架，SEO 友好

- **D3.js** - 更丰富的数据可视化
